# Support - SkyBeast Ascension

Need help with SkyBeast Ascension? We can help with account, gameplay, and technical issues.

## Contact

- Email: `support@yourdomain.com`
- Response target: within 1-3 business days

## Include This in Your Message

- Platform: Web / iOS / Android
- Device + OS version
- Game version
- Your account email
- Steps to reproduce the issue
- Screenshot or screen recording (if possible)

## Common Topics

### Account and Login
- Google login issues
- Session expired / unable to sign in
- Progress not syncing

### Gameplay
- Tier progression questions
- Ability/controls troubleshooting
- Matchmaking or multiplayer sync issues

### Billing (if enabled later)
- Subscription access status
- Upgrade not applied

## Status

If there is a known outage or maintenance event, post it on your website status section and link it here.
